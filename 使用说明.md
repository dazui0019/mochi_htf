# 使用说明

## 1. 启动
```bash
uv run python main.py
```

浏览器访问：
- 本机：`http://127.0.0.1:18765`
- 局域网：`http://<工控机IP>:18765`

## 2. 用例管理
- 在“导入测试用例”卡片选择并加载用例。
- 点击“创建用例”创建一个默认空用例，并自动进入编辑模式。
- 点击“删除用例”可删除当前选择用例（测试运行中不可删除）。

## 3. 编辑规则
- 用例加载后默认只读。
- 点击“编辑用例”进入编辑模式。
- 点击“保存用例”后自动回到只读模式。
- 仅只读模式允许开始测试。
- 只读模式会隐藏新增/删除/上下移按钮。

## 4. 编辑结构
- Item 与 Step 都是可折叠卡片。
- 默认全部折叠；保存后保留当前折叠/展开状态。
- 名称在卡片顶栏直接编辑。
- `Step.type`（Action/Verify）在顶栏编辑。
- `Step.plugin` 与 `Step.action` 为下拉选择：
  - 先选插件，再选动作。
  - 切换插件后，动作会自动校正为该插件可用动作。
- 参数区：
  - `params` 使用 JSON 文本。
  - `expr` 仅 Verify 步启用。

## 5. 新增与排序
- 编辑模式下，使用卡片式 `+` 添加 Item 或 Step。
- 顶栏 `↑/↓` 可调整顺序。
- 可删除 Item/Step。

## 6. 执行与结果
- 点击“开始测试”执行。
- 点击“停止测试”后，当前 Step 结束后停止。
- 运行状态：
  - `Pass`、`Fail`、`Error`、`Stopped`
- 结果在左侧 Item/Step 右侧实时显示。

## 7. 历史与导出
- “执行与历史”显示历史摘要。
- 点击历史中的测试用例名称（超链接）打开详情页。
- 在详情页执行导出：
  - JSON：`/api/reports/{run_id}.json`
  - Excel：`/api/reports/{run_id}.xlsx`
